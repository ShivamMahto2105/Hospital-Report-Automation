# Hospital Data Analysis Project

## Overview
This project generates and analyzes hospital patient data using Python.  
It demonstrates basic **data analysis, visualization, and report generation**.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- OpenPyXL

## Files
- `data_generator.py` – Generates the hospital dataset  
- `hospital_data.csv` – Dataset with 1200 patient records  
- `analysis_report.py` – Performs analysis and creates charts  
- `hospital_report.xlsx` – Final Excel report with charts

## Dataset Features
- patient_id
- age
- gender
- department
- diagnosis
- billing_amount
- insurance_claim
- status
- admission_date
- discharge_date
